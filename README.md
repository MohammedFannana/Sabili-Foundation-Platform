## Sabili Foundation
A comprehensive, secure, and responsive web-based management system (ERP) designed specifically for Sabili Foundation to streamline charity operations. The system functions as a centralised hub to track and manage orphan profiles, active sponsorships, employee databases, and legal attachments. It features dual-method data entry (individual forms or bulk Excel importing), a custom query builder for auditing requests, and an integrated financial ledger that tracks paid/delayed payments and exports comprehensive statements.

### Skills:
PHP - Laravel – MySQL – JavaScript – API – Data Export & Reporting – File Handling - Advanced Filter 

### Deliverables:
- Individual forms and bulk Excel import for orphan profiles. 
- Analytics Dashboard: Quick-view charts for orphans and financial stats.
- Secure tracking for paid/overdue disbursements with statement export feature.
- Multi-conditional query builder to review, approve, or reject new registrations
